# 推送代码到 GitHub 指南

## 方法一：使用 Personal Access Token（推荐）

### 步骤：

1. **生成 Personal Access Token**
   - 访问：https://github.com/settings/tokens
   - 点击 "Generate new token" → "Generate new token (classic)"
   - 填写 Note（如：lucky-wheel）
   - 选择过期时间（建议选择较长时间，如 90 天）
   - **重要**：勾选 `repo` 权限（这会自动勾选所有 repo 相关权限）
   - 点击 "Generate token"
   - **立即复制 token**（只显示一次！）

2. **使用 token 推送代码**
   
   在终端执行：
   ```bash
   git push -u origin main
   ```
   
   当提示输入用户名时：
   - Username: 输入你的 GitHub 用户名 `Goforist`
   - Password: **粘贴刚才复制的 token**（不是你的 GitHub 密码！）

## 方法二：使用 SSH（更安全，推荐长期使用）

### 步骤：

1. **检查是否已有 SSH 密钥**
   ```bash
   ls -al ~/.ssh
   ```
   如果看到 `id_rsa.pub` 或 `id_ed25519.pub`，说明已有密钥，跳到第 3 步。

2. **生成新的 SSH 密钥**
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"
   ```
   按回车使用默认设置（直接按回车，不要设置密码短语）

3. **复制公钥**
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
   或
   ```bash
   cat ~/.ssh/id_rsa.pub
   ```
   复制输出的内容

4. **添加到 GitHub**
   - 访问：https://github.com/settings/keys
   - 点击 "New SSH key"
   - Title: 填写名称（如：My MacBook）
   - Key: 粘贴刚才复制的公钥
   - 点击 "Add SSH key"

5. **修改远程仓库地址为 SSH**
   ```bash
   git remote set-url origin git@github.com:Goforist/lucky-wheel.git
   ```

6. **推送代码**
   ```bash
   git push -u origin main
   ```

---

## 方法三：使用 GitHub CLI（最简单）

如果你安装了 GitHub CLI：

```bash
# 登录 GitHub
gh auth login

# 推送代码
git push -u origin main
```

---

## 推荐方案

- **快速推送**：使用方法一（Personal Access Token）
- **长期使用**：使用方法二（SSH），一次设置，永久使用

---

## 推送成功后

代码推送成功后，按照以下步骤启用 GitHub Pages：

1. 访问：https://github.com/Goforist/lucky-wheel
2. 点击 "Settings"（设置）
3. 左侧菜单找到 "Pages"
4. 在 "Source" 下：
   - 选择 "Deploy from a branch"
   - Branch 选择 "main"
   - Folder 选择 "/ (root)"
5. 点击 "Save"
6. 等待 1-2 分钟，你的网站链接就会生成：
   **https://goforist.github.io/lucky-wheel/**
