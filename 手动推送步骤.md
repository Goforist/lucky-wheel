# 手动推送代码步骤

由于自动推送遇到权限问题，请按照以下步骤手动推送：

## 方法一：使用命令行（推荐）

在终端执行以下命令：

```bash
cd /Users/chenqinhao/CodeProject
git push -u origin main
```

当提示输入时：
- **Username**: `Goforist`
- **Password**: 粘贴你的 Personal Access Token（不是 GitHub 密码）

**注意**：如果还是出现 403 错误，可能是：
1. Token 权限不足 - 需要确保 token 有 `repo` 权限
2. Token 已过期 - 需要重新生成

---

## 方法二：检查 Token 权限

1. 访问：https://github.com/settings/tokens
2. 找到你刚才创建的 token
3. 确保它有以下权限：
   - ✅ `repo` (Full control of private repositories)
   - ✅ 或者至少 `public_repo` (Access public repositories)

如果没有这些权限，需要：
- 删除旧 token
- 创建新 token，**务必勾选 `repo` 权限**

---

## 方法三：使用 GitHub Desktop（最简单）

如果你安装了 GitHub Desktop：

1. 打开 GitHub Desktop
2. File → Add Local Repository
3. 选择 `/Users/chenqinhao/CodeProject`
4. 点击 "Publish repository"
5. 输入 token 作为密码

---

## 推送成功后

代码推送成功后，访问：
**https://github.com/Goforist/lucky-wheel/settings/pages**

启用 GitHub Pages：
1. Source 选择 "Deploy from a branch"
2. Branch 选择 "main"
3. Folder 选择 "/ (root)"
4. 点击 "Save"

你的网站链接将是：
**https://goforist.github.io/lucky-wheel/**
