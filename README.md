# 幸运转盘 🎡

一个美观、功能完整的网页转盘抽奖应用，支持自定义选项和随机抽奖。

## 功能特点

- ✨ **自定义选项**：可以添加任意数量的选项到转盘
- 🎨 **美观界面**：现代化的渐变设计和流畅的动画效果
- 🎯 **随机抽奖**：点击按钮随机旋转转盘，公平抽选
- 📱 **响应式设计**：支持桌面和移动设备
- 🗑️ **选项管理**：可以随时删除不需要的选项

## 使用方法

1. 打开 `index.html` 文件在浏览器中
2. 在"自定义选项"区域输入选项名称，点击"添加"按钮
3. 至少添加2个选项后，点击"开始抽奖"按钮
4. 转盘会自动旋转并随机选中一个选项
5. 抽奖结果会显示在转盘下方

## 技术实现

- 纯 HTML/CSS/JavaScript 实现，无需任何依赖
- 使用 Canvas 绘制转盘
- CSS3 动画实现流畅的旋转效果
- 响应式布局适配各种屏幕尺寸

## 浏览器支持

支持所有现代浏览器（Chrome、Firefox、Safari、Edge 等）
