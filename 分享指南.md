# 转盘应用分享指南

有几种方式可以分享你的转盘应用：

## 方法一：GitHub Pages（推荐）⭐

这是最专业且免费的方式，可以生成一个在线链接分享给任何人。

### 步骤：

1. **创建 GitHub 仓库**
   - 访问 [GitHub](https://github.com) 并登录
   - 点击右上角 "+" → "New repository"
   - 仓库名：`lucky-wheel`（或任意名称）
   - 选择 Public（公开）
   - 点击 "Create repository"

2. **上传文件**
   - 在仓库页面，点击 "uploading an existing file"
   - 将 `index.html` 文件拖拽上传
   - 点击 "Commit changes"

3. **启用 GitHub Pages**
   - 在仓库页面，点击 "Settings"（设置）
   - 左侧菜单找到 "Pages"
   - 在 "Source" 下选择 "main" 分支
   - 点击 "Save"
   - 等待几分钟，会生成一个链接，格式如：`https://你的用户名.github.io/lucky-wheel/`

4. **分享链接**
   - 将这个链接分享给任何人，他们就可以直接访问你的转盘应用了！

---

## 方法二：Netlify Drop（最简单）🚀

无需注册，拖拽即可上线！

1. 访问 [Netlify Drop](https://app.netlify.com/drop)
2. 将 `index.html` 文件拖拽到页面
3. 立即获得一个在线链接（如：`https://random-name-123.netlify.app`）
4. 分享这个链接即可

---

## 方法三：直接发送文件 📧

最简单但需要对方下载：

1. 将 `index.html` 文件通过微信、QQ、邮件等方式发送
2. 对方下载后双击打开即可使用
3. **注意**：这种方式对方需要下载文件，不如在线链接方便

---

## 方法四：使用在线代码平台

### CodePen
1. 访问 [CodePen](https://codepen.io)
2. 注册/登录账号
3. 点击 "Create" → "Pen"
4. 将 `index.html` 中的代码分别复制到 HTML、CSS、JS 区域
5. 点击 "Save" 并设置为 "Public"
6. 分享链接

### JSFiddle
类似 CodePen，访问 [JSFiddle](https://jsfiddle.net) 进行操作

---

## 推荐方案对比

| 方式 | 难度 | 优点 | 缺点 |
|------|------|------|------|
| GitHub Pages | ⭐⭐ | 专业、免费、可更新 | 需要 GitHub 账号 |
| Netlify Drop | ⭐ | 最简单、无需注册 | 链接可能不够个性化 |
| 直接发送文件 | ⭐ | 最简单 | 需要下载，不够方便 |
| CodePen/JSFiddle | ⭐⭐ | 在线编辑 | 需要注册，界面可能不够简洁 |

**最推荐：GitHub Pages** - 专业、稳定、免费，适合长期使用和分享！
